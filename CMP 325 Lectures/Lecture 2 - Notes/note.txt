What is a Stack Data Structure?
A stack data structure is a data structure that follows LIFO Principle. (Last In First Out)

Give us an example use case of this data structure in a real life scenario
- Stacking of dishes
- Opening Google Chrome tabs


Operations performed with stacks
- Insert (Push)
- Remove (Pop)
